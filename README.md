# 🌿 Garden Match Masters

**Garden Match Masters** is a polished match-3 garden arcade game built for GitHub Pages.  
The goal is simple: swap neighbour tiles, create matches, collect garden pieces, and complete the mission before running out of moves.

🔗 **Live demo:** https://laurandreea10.github.io/Garden-Match-Masters/

---

## ✨ Features

- 🌼 Match-3 board with flowers, fruit, leaves, mushrooms and carrots
- 🎯 Mission progress system
- 🔁 Combo multiplier
- 🧠 Hint button
- 🏆 Local best score saved in the browser
- 🌍 Bilingual interface: Romanian / English
- 📱 Responsive layout for mobile and desktop
- ♿ Keyboard focus styles and reduced-motion support
- 🚀 GitHub Pages ready with `index.html`

---

## 🎮 How to play

1. Select two neighbour tiles.
2. If the swap creates a line of 3 or more matching items, the tiles disappear.
3. New garden pieces fall into the board.
4. Build combos to increase your score.
5. Complete the mission before the moves reach zero.

## 🚀 Deploy on GitHub Pages

This repository is ready for GitHub Pages because it contains an `index.html` file in the root.

Recommended GitHub Pages settings:

```txt
Source: Deploy from a branch
Branch: main
Folder: /root
```

After saving the settings, the project should be available at:

```txt
https://laurandreea10.github.io/Garden-Match-Masters/
```
